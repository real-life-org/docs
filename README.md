## Abgrenzung

**Organisation** (real-life.org) = Das Kernteam, das die Werkzeuge baut und pflegt. Mit RFCs, Rollen, klarer Mitgliedschaft.

**Netzwerk** (reallife.network) = Alle Menschen und Initiativen, die sich verbinden. Kein Beitritt nötig, kein Zentrum, gehört niemandem.

Die Organisation dient dem Netzwerk. Sie baut die Infrastruktur — kontrolliert aber nicht das Netzwerk.
